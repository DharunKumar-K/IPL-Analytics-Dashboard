# Cricket Analytics Dashboard

Interactive Streamlit dashboard for IPL cricket analytics with advanced visualizations.

## Features
- Player performance analysis (batting & bowling)
- Team statistics and win percentages
- Venue performance metrics
- Interactive charts and trends

## Setup
```bash
pip install -r requirements.txt
streamlit run scripts/cricket_dashboard.py
```

## Data
Place your cricket data file as `data/combined_player_match_s_format.csv`

## Usage
Run the dashboard:
```bash
cd scripts
streamlit run cricket_dashboard.py
```
